<h1>Jogo do Número Secreto</h1>

<h2>📋 Sobre</h2>
<p>Um jogo interativo onde o jogador tenta adivinhar um número secreto gerado aleatoriamente. O jogo fornece dicas indicando se o número a adivinhar é maior ou menor que o palpite do jogador.</p>

<h2>🎮 Como Jogar</h2>
<p>1. Digite um número no campo de entrada</p>
<p>2. Clique no botão "Chutar"</p>
<p>3. Receba dicas se o número secreto é maior ou menor</p>
<p>4. Continue tentando até acertar!</p>
